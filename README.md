# Random GitHub Repo

A web application that finds random GitHub repositories based on random words.

https://wxn0brp.github.io/random-git

## Features

- Fetches random words from a words API
- Searches GitHub repositories using those words
- Filter by programming language
- Pagination support (page & per-page options)
- Displays repository info: stars, forks, description, last update

## Tech Stack

- **APIs:**
  - Random Words API
  - GitHub Search API
